# 玩家操作:
* 上鍵: 方塊順時針選轉
* 左鍵: 方塊左移
* 右鍵: 方塊右移
* 下鍵: 方塊向下
* Esc: 遊戲暫停
* Enter: 重新開始
* space: 方塊直接下降到底層
